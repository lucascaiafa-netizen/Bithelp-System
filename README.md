# Bithelp

Projeto de TCC desenvolvido como uma solução de automação e suporte técnico, focada em otimização de fluxo de trabalho e gestão eficiente de chamados.

## 🚀 Sobre o Projeto
O Bithelp foi criado para resolver problemas comuns de organização no suporte técnico. A aplicação utiliza uma arquitetura baseada em containers (Docker), garantindo que o ambiente de execução seja idêntico em qualquer máquina, facilitando tanto o desenvolvimento quanto o deploy.

## 🛠 Tecnologias Utilizadas
- **Python:** Linguagem principal para a lógica de negócio.
- **Docker & Docker Compose:** Utilizados para conteinerizar a aplicação e facilitar a configuração do ambiente de desenvolvimento.
